# CSS Box Shadow Examples

A simple project demonstrating different CSS `box-shadow` effects using HTML and CSS.

## Features

- Outer shadows
- Inset shadows
- Different shadow offsets
- Various blur and spread values

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.

## Purpose

This project was created to practice and understand the CSS `box-shadow` property and its different effects.
